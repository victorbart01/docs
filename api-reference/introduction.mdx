## Bienvenue sur View4Sight 👋

View4Sight est une plateforme collaborative de visualisation et d’analyse 3D pensée pour les géomètres, urbanistes, collectivités et ingénieurs.  
Notre objectif : vous permettre d'explorer, mesurer et partager vos projets 3D de façon simple, rapide et intuitive.

## Ce que vous trouverez dans cette documentation

La documentation est divisée en plusieurs sections pour vous accompagner dans votre prise en main :

- 🧭 **Quick Start** : découvrez comment importer un projet et naviguer dans l’interface.
- 🛠️ **Tutoriels spécifiques** : explorez des cas d’usage concrets (contrôle de planéité, annotations, vues partagées...).
- 🎥 **Tutoriels vidéo** : accédez à des démonstrations filmées pas à pas.
- 🧪 **Référence API** *(à venir)* : bientôt, vous pourrez intégrer View4Sight à vos outils via notre API.
